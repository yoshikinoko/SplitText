# SplitText

[Adobe XD](https://www.adobe.com/products/xd.html)插件，用于按换行符，制表符和逗号分隔文本。

- 选择文字元素
- `Plugins`>`在换行符上分割文本`以在换行符上分割所选文本

## 说明
按换行符，制表符和逗号分隔文本。
(要通过选项卡拆分文本，请从另一个编辑器复制并粘贴文本。)

## 如何安装此插件

1. 启动XD并转到 `插件` > `发现插件`
2. 搜索"SplitText"
3. 单击安装
